# Agency Project

## Purpose
This folder is the operating space for my agency.

## How to work here
- Use the brand skill suite in `../../reference/` as the methodology source.
- Do not duplicate the skills here unless necessary.
- Store outputs here, not the skill definitions.

## Core workflow
1. `brand-dna` for identity foundations
2. `brand-os` for positioning, strategy, offer logic, and development
3. `scene-architect` for visual prompt production when needed

## Folder rules
- `brand/` → positioning, naming, tone, archetype, brand foundations
- `website/` → sitemap, copy, structure, Framer/code files
- `offers/` → service design, packages, pricing, proposals
- `operations/` → process, onboarding, delivery workflow
- `content/` → authority content, campaign ideas, editorial direction
- `assets/` → logos, images, exports, visual materials
- `notes/` → drafts, raw thinking, research, temporary material

## Output preference
- Keep strategy clear and practical
- Avoid generic branding language
- Prefer structured outputs that can be turned into execution
