# Typography

Consistent typography reinforces the otma brand across digital and print.

## Typefaces

| Role | Typeface | Fallback |
|---|---|---|
| Display / Headlines | **Inter** | sans-serif |
| Body | **Inter** | sans-serif |
| Monospace / Code | **JetBrains Mono** | monospace |

## Type Scale

| Level | Size | Weight | Line Height | Usage |
|---|---|---|---|---|
| H1 | 48px / 3rem | 700 | 1.15 | Page titles |
| H2 | 36px / 2.25rem | 700 | 1.2 | Section headings |
| H3 | 24px / 1.5rem | 600 | 1.3 | Sub-section headings |
| Body | 16px / 1rem | 400 | 1.6 | Default body copy |
| Small | 14px / 0.875rem | 400 | 1.5 | Captions, labels |
| Code | 14px / 0.875rem | 400 | 1.6 | Inline and block code |

## Usage Rules

- Use **bold (700)** sparingly to highlight key information.
- Avoid using more than two type sizes in a single UI component.
- Maintain a minimum body font size of **16px** for readability.
