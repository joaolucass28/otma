# Colors

The otma color palette is designed to be clean, modern, and consistent across all touchpoints.

## Primary Palette

| Name | Hex | Usage |
|---|---|---|
| otma Black | `#0D0D0D` | Primary text, backgrounds |
| otma White | `#FFFFFF` | Backgrounds, reversed text |
| otma Accent | `#2563EB` | CTAs, links, highlights |

## Secondary Palette

| Name | Hex | Usage |
|---|---|---|
| otma Gray 100 | `#F5F5F5` | Light backgrounds, cards |
| otma Gray 400 | `#9CA3AF` | Secondary text, borders |
| otma Gray 700 | `#374151` | Body text on light backgrounds |

## Usage Rules

- Always maintain sufficient contrast (minimum **4.5:1** for body text per WCAG AA).
- Do not use the accent color as a large background area.
- Never distort or mix brand colors with unapproved tints.
