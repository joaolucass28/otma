# Logo

This folder is the home for all official otma logo files and usage guidelines.

## Files

> Place logo files in this folder. Recommended formats and naming:

| File | Format | Usage |
|---|---|---|
| `otma-logo-primary.svg` | SVG | Default — use on white/light backgrounds |
| `otma-logo-reversed.svg` | SVG | Use on dark/color backgrounds |
| `otma-logo-mark.svg` | SVG | Icon-only mark (no wordmark) |
| `otma-logo-primary.png` | PNG (2×) | Raster fallback for documents/email |

## Usage Rules

### ✅ Do
- Use the official SVG files from this folder.
- Maintain the minimum clear space: **equal to the height of the "o" in otma** on all sides.
- Scale the logo proportionally.

### ❌ Don't
- Stretch, skew, or rotate the logo.
- Change the logo colors outside of approved variants.
- Place the logo on a busy or low-contrast background.
- Recreate the logo using text/fonts.

## Minimum Size

| Medium | Minimum Width |
|---|---|
| Digital | 80px |
| Print | 20mm |
